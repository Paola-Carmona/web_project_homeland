# Tripleten web_project_homeland

# Tecnologías utilizadas

Esta página web fue diseñada con ayuda de HTML y CSS.

# Introducción

La página "De patria a patria" es un proyecto cuya intención es mostrar la multinacionalidad de aquellos que han pasado por TripleTen mediante la 'TripleTen Art Gallery'. Es una página para unir a todos sus miembros sin importar su origen.

# Contenido

La página se divide de la siguiente manera:

1. Header - cuenta con el logo de la galería de arte, el título y subtítulo del proyecto y una imagen.
2. Main - incluye otro título, una cita y una pequeña descripción de la página.
3. Copyright - derechos de autor.
